iOS 패션 브랜드 탐색 및 관리 앱

## 💡 한 줄 소개
> **좋아하는 브랜드만 쏙쏙 골라 탐색하고 관리하는 iOS 맞춤형 브랜드 탐색 앱**

패션 브랜드가 너무 많아서 정리하기 힘든 사람들을 위해  
좋아하는 브랜드를 스크랩하고, 위젯으로 확인하며  
내 스타일에 맞는 브랜드를 추천받는 스마트한 브랜드 매니저 앱입니다.

---

## ❓ 문제

- 내가 좋아하는 브랜드를 찾기 어렵다  
- 찾은 브랜드를 나중에 또 보고 싶은데 기억이 안 난다  
- 위젯으로 예쁜 브랜드를 확인하고 싶다  
- 내가 좋아하는 장르 위주로만 보고 싶다

---

## ✅ 솔루션

1. ✨ 브랜드를 장르/정렬/탑10으로 탐색 가능  
2. 💾 스크랩 기능으로 브랜드 저장 및 관리  
3. 📱 위젯으로 추천 브랜드를 주기적으로 확인  
4. 📩 로그인, 온보딩으로 나만의 패션 스타일 세팅  
5. 👤 스크랩 수에 따라 캐릭터가 성장하며 동기부여 제공

## 🛠 개발 환경

- Language: `Swift`, `Java`
- iOS Framework: `SwiftUI`
- Backend Framework: `Spring Boot`
- Database: `MySQL`
- ORM: `JPA`
- Deploy: `AWS EC2`, `Route 53`

## 🧩 주요 기능 구성

| 화면 | 주요 기능 |
|------|-----------|
| Splash | 애니메이션 효과로 인트로 표시 |
| Login | Google 로그인, 자동 세션 |
| OnBoarding | 닉네임 및 장르 설정 |
| Main | 브랜드 추천, 스크랩 제외, 배너 효과 |
| Search | 최근 검색어, 검색 결과 표시 |
| Brand | 상세 정보, 스크랩 토글, 필터 |
| Scrape | 카드 플립, 삭제, 스크랩 리스트 |
| User | 스크랩 수에 따른 캐릭터 성장 |
| History | 최근 본 브랜드 리스트 |
| Edit | 프로필 수정, 로그아웃 |
| Widget | 위젯 자동 갱신 (10초마다 추천) |

## 🧠 핵심 기능 설명

### 1. 메인 페이지
- 장르 기반 추천
- 탑10 리스트 제공
- 서버 연동 실시간 반영

### 2. 브랜드 상세
- 상세 정보 조회 및 스크랩
- 캐릭터 변화로 시각적 재미 추가
- 스크랩 수에 따라 레벨 업

### 3. 위젯 추천
- 위젯에 자동으로 추천 브랜드 반영
- 10초마다 추천 브랜드 갱신
- 시각적 동기부여 강화

### 4. 사용자 관리
- 로그인 시 세션 유지
- 닉네임, 장르 기반 맞춤 추천
- 유저 정보 조회/수정 API 연동
